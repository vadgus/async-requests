## install

#### system packages

```bash
apt install python3.7
apt install python3-pip
```

#### virtualenv
```bash
pip3 install virtualenv
virtualenv --python=python3.7 .venv
. .venv/bin/activate
```

#### dependencies
```bash
pip3 install -r requirements.txt
```

## run
```bash
python3.7 examples/main.py
```
