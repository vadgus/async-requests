# async-requests

### sending many async requests
<br>



## install

#### system packages

```
apt install python3.7
apt install python3-pip
```

#### virtualenv
```
pip3 install virtualenv
virtualenv --python=python3.7 .venv
. .venv/bin/activate
```

#### dependencies
```
pip3 install -r requirements.txt
```
<br>

## run
```
python3.7 main.py
```
